## 7. Связи между механиками (web)

```
Бой-ядро ──┬─ Эскадрильи ─── counter: aaDamage
           ├─ Ракеты ─────── counter: pointDefense ; детект: signature/радар
           ├─ Щиты(корпус+щит) ── ремонт ─┬─ Порт
           │                              ├─ Ремонтные дроны ─ это вариант Эскадрильи (in-combat хил)
           │                              └─ Ремонтный модуль ─ из Экономики (F2P-catchup)
           └─ Захват/орбита

Async ──┬─ Экономика (offline-accrual)
        └─ Цепочки команд ── offline-будилка (PA-4.1) ; стоячие приказы (патруль эскадрилий)

Мета ──┬─ Аккаунт → Уровень → гейт(корп-создание, аукцион)
       ├─ Корпорации → роли/медали(→витрина в меню) → AvA(слоты/команды карты)
       ├─ AvA → лента новостей (меню) ; награды → влияние/рейтинг корп
       └─ Экономика/P2W → инвентарь модулей ─→ Лоадаут-фундамент (ModuleDef/слоты/эфф-статы)
                                              ├─ трюм-расширитель (+N) ; фабрика дройдов (мобильн. сборка)
                                              └─ плагины: пусковой ракет ; ремонт ; реген-щита ; лоадаут эскадрилий — с F2P-catchup, tag horizontal|vertical
```

Ключевая синергия: **щиты ↔ ремонт ↔ дроны ↔ эскадрильи** (каждая даёт остальным смысл);
**ракеты ↔ pointDefense ↔ signature/радар**; **цепочки команд ↔ offline-первичность** (главный столп).

## 8. Предлагаемый порядок сборки

1. **Карта-слоты/команды** (✅) → разблокирует AvA/матчмейк.
2. **Щиты** (касается ЯДРА — golden) → затем **эскадрильи** → **ремонтные дроны** → **ракеты**
   (боевой контент на готовом ядре).
3. **Цепочки команд** (на offline-будилке) — async-защита.
4. **Мета по зависимостям:** аккаунты → уровень → корпорации → AvA → экономика/P2W → меню/клиент.
5. **Инфра параллельно** (сервер/персист/безопасность) — предусловие всего онлайна.

## 9. Сквозные инварианты (каждая механика обязана)

- **Детерминизм** (нет `Date.now`/`Math.random` в ядре; всё через `schedule`/хуки/события).
- **Только модуль + данные, кернел не трогаем** — исключение: щиты (правят `applyDamage`, осознанный golden).
- **Server-authority + fail-secure** (клиент шлёт намерение; отказ со стабильным кодом).
- **Async/offline-first** (мир идёт 24/7; механики исполняются без клиента).
- **P2W-honesty-guardrails** (силовой контент: F2P-catchup, раскрытие шансов, soulbound донат/лутбокс, гейт уровнем; контр-игра доступна всем).
- **Горизонталь, не вертикаль** для прогресса (§5.2) — числовая ветвь по премиум-неускоряемой оси.
- **Туман/видимость** (контр-игра требует видеть угрозу — напр. ракету по signature).
